    • [后端] 增加作者注册、小说/章节发布等部分作家相关接口

    • [定位] 产品方向调整为学习型 JAVA 项目
    • [架构] 遵循阿里巴巴最新开发规范（嵩山版），数据库重新设计，项目结构调整
    • [技术] 使用当前最新技术栈 Spring Boot 3 + Vue 3 开发，项目驱动式学习，并定期升级
    • [功能] 上线 novel 后端项目和 novel-front-web 前端前台门户系统项目
    • [文档] 上线项目实战教程 - 手把手教你从零开始开发上线一个生产级别的小说系统 (opens new window)
